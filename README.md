# 🎓 Student Result System (Python)

A simple menu-driven Student Result Management System built using Python. This project was created to practice Python fundamentals such as functions, lists, dictionaries, loops, conditionals, and problem-solving.

## ✨ Features

* ➕ Add Student
* 👀 View All Students
* 🔍 Search Student by Roll Number
* 🏆 Find Class Topper
* 📊 Class Statistics

  * Highest Marks
  * Lowest Marks
  * Average Marks
  * Passed Students Count
  * Failed Students Count
* 📝 Automatic Grade Calculation
* 🔄 Menu-Driven Interface

## 🛠️ Concepts Used

* Python Functions
* Parameters & Return Values
* Lists
* Dictionaries
* List of Dictionaries
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`for`, `while`)
* Boolean Flags
